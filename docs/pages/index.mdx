# Cortex Documentation

Cortex is a persistent memory layer for Claude Code. It runs as a local MCP server, captures decisions, context, and preferences during your coding sessions, and injects them back into every future session automatically.

## 30-Second Install

```bash
npx @cortex.memory/cli init
```

That's it. Cortex registers itself as an MCP server with Claude Code, creates a local SQLite database, and starts the daemon. Your next Claude Code session will have persistent memory.

Verify the install:

```bash
cortex doctor
```

Expected output:

```
✓ Daemon running (PID 12847)
✓ MCP server registered with Claude Code
✓ Database exists (~/.cortex/cortex.db)
✓ Port 7434 available
✓ Dashboard available at http://localhost:7433
All checks passed. Cortex is healthy.
```
